# An example of aiogram bot

## How to run
* Install and run Docker
* Build Docker image using `docker build . -t server`
* Run Docker container using `docker run --rm -it _server`
* Go to `telegram` to see all 

**All parameters are hardcoded to make the example as easy as possible**

* contains API logic
